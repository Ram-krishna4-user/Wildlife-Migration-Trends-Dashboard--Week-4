# Wildlife-Migration-Trends-Dashboard--Week-4
#  Animal Migration Analytics Dashboard   
### Built using Power BI, Tableau Public, Excel | Data Visualization Project

This project analyzes **animal migration patterns** using GPS movement data, focusing on  
migration routes, total distances, speed behavior, regional activity, and species profiles.

It includes **two full dashboards**:  
- **Power BI Interactive Dashboard**  
- **Tableau Public Dashboard**

---

##  Project Overview

The objective of this project is to visualize and understand wildlife movement using  
latitude–longitude tracking data combined with species information.

###  Key Metrics (KPIs)
- Total Animals Tracked  
- Total Migration Distance  
- Average Movement Speed  
- Total Movement Records  
- Region-wise Migration  
- Animal-wise Distance Summary  

###  Key Visualizations
- Migration Path Map (GPS Line Paths)  
- Density Map (Movement Hotspots)  
- Distance Trend Over Time  
- Speed vs Distance Scatter  
- Region-wise Movement Analysis  
- Animal Profile Summary Table

##  Dataset Description

Two CSV files were used:

### **1. animals.csv**
Contains animal attributes:
- Animal ID  
- Species  
- Age  
- Region  


### **2. animal_movements.csv**
Contains GPS tracking points:
- Timestamp  
- Latitude  
- Longitude  
- Previous Lat/Long (Calculated)  
- Distance (Km)  
- Speed (Km/h)  

Excel formulas were used to calculate **distance** and **speed** using sequential GPS points.
